# SuperMovies2

This app shows list of movies from https://api.themoviedb.org/3/ using
the API they provide.

I used:

* Android framework.
* Kotlin.
* Retrofit.
* Coroutines.
* Dependency injection (Hilt).
* Clean code.
* MVVM.
* Lifecycle.
* Viewmodel.
* RecyclerView.
* View binding.
* Glide.
* Material library.
* Fragments.
